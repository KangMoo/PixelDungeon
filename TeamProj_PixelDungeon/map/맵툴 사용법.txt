f1 = 타일 입력, f2 = 데코레이션 입력, f3 = 오브젝트 입력, f4 = 아이템 입력(안씀), f5 = 몬스터 입력 
각 모드에 대해 1,2,3,4,5,6 등을 누르면 무엇을 입력할 건지 설정 가능

ctr + o = 로드, ctr + s = 세이브, 마우스 오른쪽 클릭 = 타일 범위지정, ctr + d 선택타일 취소

123456 이건 여러번 누르면 모드가 바뀌기도 하기 이것저것 눌러보고

0은 읽기 모드(깔아놓은 타일 구경), 0 한번 더 누르면 타일의 성질을 색으로 확인 가능

e = 지우개, delete = 선택 범위 삭제 (참고로 모드 설정한 영역만 삭제 가능. 예를들어 f1 누른 상태면 타일만 삭제)

ctrl 누른 상태에서 범위 선택하면, 기존에 선택한 범위 유지한 상태에서 범위 추가 가능 

선택한 범위에 타일 입력하면 범위 전체에 타일 깔림
